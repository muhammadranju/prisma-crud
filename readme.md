## Prisma CRUD

This is a CRUD application that uses Prisma ORM to connect to MongoDB.

## Getting Started

### 1. Install Dependencies

```bash
npm install
```

### 2. Start the Server

```bash
npm run dev
```

### 3. Open the App

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.
